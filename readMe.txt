My app is a simple Drum Machine, you can set bpm, and program sequence for 4-drum kit. Choose a drum, program sequence for it 
by checking/uncheching togglebuttons (ToggleButtons Represent step from 1 - 16), and press "banana" to play or brokkoli to stop.
Long click bpm "+"/"-" buttons to add bpm value by +/- 10.

see more details in manual.pdf
