A simple Drum Machine, you can set bpm and program sequence for 4-drum kit. Choose a drum, program sequence for it 
by checking/unchecking togglebuttons (ToggleButtons Represent step 1 - 16), and press "banana" to play or broccoli to stop.
Long click bpm "+"/"-" buttons to add bpm value by +/- 10.

see more details in manual.pdf
